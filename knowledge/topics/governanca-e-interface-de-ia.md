# Governança e Interface de IA

4 notes convergem na tese de que o debate sobre qual LLM usar obscurece as perguntas mais importantes: em qual canal? Com qual governança? Com qual intencionalidade?

## Tese central

[Missing Interface](../notes/missing-interface-ai.md) nomeia o problema: a interface é a dimensão negligenciada. [Agentic AI](../notes/agentic-ai.md) oferece o framework prático (one-way/two-way door). [Sapia](../notes/sapia-ia-america-latina.md) implementa o princípio de aprovação humana para ações transacionais. [ROI em IA](../notes/roi-em-ia.md) mostra o custo financeiro da ausência de governança: é o gargalo central para o ROI organizacional.
