# Futuro do Trabalho e do Developer

4 notes abordam o mesmo fenômeno de ângulos complementares: o que um profissional precisa ser, saber e fazer na era da IA.

## Convergência central

[AI is not a tool](../notes/ai-not-tool.md) define o trabalhador metacognitivo — problem designer, orquestrador de agentes, avaliador crítico — como o perfil irreduzivelmente humano do século XXI. [O Novo Dev](../notes/o-novo-dev.md) concretiza isso no produto engineer: bases técnicas sólidas + visão de produto + acesso a dados. [Cloud Code norms](../notes/cloud-code-norms.md) mostra como um time real reorganizou ownership, reviews e planejamento em torno disso. [Agentic AI](../notes/agentic-ai.md) argumenta que pessoas com agentes substituem pessoas sem agentes.

## Notas neste tópico

- [AI is not a tool](../notes/ai-not-tool.md) — triadic intelligence, metacognitive worker, work shifting from programmed execution
- [O Novo Dev](../notes/o-novo-dev.md) — product engineer profile, mercado de trabalho, não-devs abrindo PRs
- [Agentic AI](../notes/agentic-ai.md) — pessoas+agentes substituem pessoas sem agentes; democratização para áreas de negócio
- [Cloud Code team norms](../notes/cloud-code-norms.md) — code ownership questions, human review reservado para legal/security/architecture
